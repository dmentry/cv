Профессиональные навыки

  • Ruby
    • ООП, классы: понимание принципов ООП, важности code style, принципов инкапсуляции, наследования, полиморфизма, абстракции
    • Установка Ruby: использование rvm или rbenv
    • Работа с форматами: XML, JSON, XLSX, DOC
  • Ruby on Rails
    • Основы проектирования приложений: изначально необходимо корректно поставить задачу, продумать сущности, написать сценарии и создать CRUD
    • MVC: основа архитектуры приложений
    • Active Record: создание ассоциаций между моделями, построение запросов к базе данных с помощью интерфейса запросов, создание миграций, используя генератор, написание валидаций и коллбэков
    • Action Mailer: отправка почты с помощью Mailjet. В development–окружении использование гема Letter Opener
    • Аутентификация пользователя с помощью Devise и его авторизация, используя Pundit или Cancancan. Для доступа с учетными данными соц. сетей использование OmniAuth
    • Credentials: хранение паролей, ключей и иной конфиденциальной информации
    • Rake: создание задач и использование cron
    • State Machines - конечный автомат для моделей
    • Ransack - поиск
    • Carrierwave, Rmagick, Fog::Aws – использование для загрузки, обработки и хранения изображений (и не только) в проекте
    • Webpacker для интеграции Rails со сборщиком ассетов webpack
    • Pry для дебага (и в Ruby тоже)
    • Recaptcha – защита от спама с помощью технологии Google reCAPTCHA
  • Тестирование
    • RSpec: по возможности, весь основной код необходимо покрывать тестами. Работал с гемом FactoryBot. Для интеграционных тестов использовал Capybara
  • Базы данных
    • Опыт работы с SQLite, MySQL, PostgreSQL
  • Deploy
    • VPS: установка веб-серверов Nginx и Phusion Passenger c PostgreSQL. Деплой Rails–приложения с помощью Capistrano
    • Heroku: развертывание приложения
  • Технологии
    • Git, Github: базовое использование
    • Linux: Ubuntu
    • RubyMine, Sublime: написание кода
    • AWS S3: хранение файлов
  • Frontend
    • HTML, CSS, Bootstrap4, JS, jQuery, ExtJS, Slim
  • Иностранный язык
    • Английский: upper intermediate
